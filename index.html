Widget World

Widget World is a personal dashboard with a customizable avatar using the Bitmoji API to represent themselves as a fun little feature.

Users have access to a calculator, clock, weather forecast, random joke generator, a trivia game in the form of multiple choices, and a product search using Walmart's API.

Widget World is built with Node.js and React, along with MongoDB as the database program

Quisine

A recipe search website utilizing the Yummly Database API that display ingredients, nutritional information, and hyperlinks to directions for cooking. The main page displays trending recipes and users can choose a category of cuisine to display as well
